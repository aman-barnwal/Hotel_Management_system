## Team

## Description
This is a console-based Hotel Reservation System written in C.  
The program allows users to book rooms, view bookings and checkout guests.  
All customer data is stored permanently using file handling.

The project demonstrates the use of:
- structure
- pointer
- functions
- arrays
- file operations

---

## Features
- Book room
- Prevent double booking
- View all reservations
- Checkout system
- Bill calculation
- Data stored in file permanently

---

## Concepts Used

### Structure
Stores guest details.

### Pointer
Used to pass structure to functions.

### Array
Used to store all records while performing operations.

### File Handling
Used to store and retrieve data from `hotel.dat`.

### Functions
Code is divided into reusable units.
