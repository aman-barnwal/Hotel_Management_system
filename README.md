## Description
This project is a console-based Hotel Management System developed in the C programming language.  
The system allows users to book rooms, view reservations, checkout guests, and calculate bills using file handling.

The project uses:
- structure
- pointer
- arrays
- file handling
- functions

---

## Working Explanation

Booking System  
User enters room number, name, phone, and number of days. Room availability is checked before saving.

View System  
All bookings are read from file and displayed using arrays.

Checkout  
Room record is removed and final bill is displayed.
